# vpsbox

VPS 初始化、系统优化与 sing-box 节点管理脚本。

当前版本：`v1.0.39`

## VPS 一键安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/TianPingXi/vpsbox/main/vpsbox.sh)
```

## 管理命令

安装完成后，可随时输入以下命令打开管理面板：

```bash
vpsbox
```
